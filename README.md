# 🤖 AI AGENTE HUB

Minha equipe pessoal de agentes de IA especializada em desenvolvimento de software, arquitetura, DevOps, aprendizado contínuo e crescimento profissional.

O objetivo deste repositório é reunir agentes especializados que atuam como uma equipe virtual de engenharia, ajudando a desenvolver raciocínio técnico, autonomia, capacidade de resolução de problemas e tomada de decisões arquiteturais.

---

# 🎯 Missão

Evoluir de um profissional que utiliza ferramentas para um profissional capaz de:

* Projetar sistemas
* Resolver problemas complexos
* Diagnosticar falhas
* Tomar decisões arquiteturais
* Liderar soluções técnicas
* Crescer como Engenheiro de Software e DevOps Engineer

---

# 🧠 Filosofia

Todos os agentes seguem os mesmos princípios:

* Compreender antes de agir
* Questionar premissas
* Explicar o raciocínio
* Ensinar quando possível
* Dividir problemas complexos em etapas
* Explicar trade-offs
* Priorizar clareza e precisão
* Seguir boas práticas de engenharia
* Seguir boas práticas de Prompt Engineering da OpenAI
