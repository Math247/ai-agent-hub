# Fluxos Recomendados de Agentes

## Nova Funcionalidade

Senior Engineering Mentor
↓
Systems Architect
↓
Pair Programmer
↓
Senior Code Reviewer

Use este fluxo quando o usuário precisar transformar uma ideia ou requisito em uma funcionalidade implementada com qualidade.

## Novo Projeto

Prompt Engineer
↓
Senior Engineering Mentor
↓
Systems Architect
↓
Pair Programmer
↓
Senior Code Reviewer

Use este fluxo quando o usuário estiver começando um projeto do zero e precisar estruturar visão, arquitetura, implementação e revisão.

## Problema em Produção

Bug Investigator
↓
Systems Architect
↓
Pair Programmer
↓
Senior Code Reviewer

Use este fluxo quando houver incidente, bug complexo, degradação de performance ou falha em ambiente produtivo.

## Estudos

Professor de TI e Engenharia de Software
↓
Senior Engineering Mentor
↓
Projetos Práticos

Use este fluxo quando o usuário quiser aprender, revisar fundamentos, criar roadmap de estudo ou se preparar para entrevistas.

## Carreira

DevOps/SRE Career Advisor
↓
Senior Engineering Mentor
↓
Plano de Evolução

Use este fluxo quando o usuário quiser planejar crescimento profissional, mudar de cargo, buscar vagas, evoluir salário ou fortalecer posicionamento.
